Tutorials, Referenzen & Co.

Codecademy: Coden lernen Schritt für Schritt in Online-Kursen, mit geführten Selbstversuchen direkt im Browser. Für HTML, Javascript, jQuery und viele andere Programmiersprachen.
Link

Codepen: Viele Code-Beispiele für Webanwendungen aus allen möglichen Bereichen. Jeweils HTML, CSS direkt im Browser schreibbar und auch direkt ausführbar.
Link

w3schools: Umfangreiche Referenz-Sammlungen, Beispiele und Tutorials für HTML, Javascript, CSS.
Link

stackoverflow: Forum zur Diskussion von spezifischen Problemlösungen. Hauptsächlich für Webanwendungen, aber auch für anderes Programmierte. Oft findet man dort die Lösung für das eigene Problem, das andere schonmal so oder ähnlich hatten.
Link

selfHTML: Der Dinosaurier unter den deutschsprachigen Referenz-Seiten zum Thema Webprogrammierung. Mit Dokumentations-Wiki incl. umfangreicher HTML-Referenz, Forum etc.

Werkzeuge

Eine kleine Liste von Programmen, die bei der Webentwicklung essentiell oder hilfreich sind.

Atom: Guter, einfacher Code-Editor for free, mit einer Menge Plugins erweiterbar und individuell anpassbar. Entwickelt von Github.
Link

Brackets: Einfacher aber guter Code-Editor mit vielen möglichen Plugins. Ehemals Adobe CC, mittlerweile selbständig. Open Source.
Link

Sublime Text: Eine andere gute Lösung zum Code schreiben. Ebenfalls viele Plugins etc. möglich.
Link

MAMP: Virtueller Webserver zur lokalen Installation, incl. PHP und MySQL-Datenbanken. Früher nur OS X, mittlerweile auch für Windows verfügbar.
Link

XAMPP: Virtueller Apache-Server zur lokalen Installation, incl. PHP und MySQL. Open Source und für alle Plattformen.
Link

FileZilla: Programm zur (FTP)-Übertragung von Daten auf entfernte Server, z.B. auf Webserver.
Link

GitHub: Online Projekthosting für die Versionierung und zur gemeinsamen Arbeit an Projekten. Wird auch oft benutzt, um Open-Source-Programme und -Bibliotheken etc. zu hosten.
Link

CSS

"Cascading Style Sheets" sind Anweisungen zur Gestaltung der Elemente einer Website.

W3Schools: CSS Tutorial: Tutorial zum Thema "CSS" (englisch)
Link

SelfHTML CSS-Referenz: CSS-Referenz auf deutsch auf der Seite "SelfHTML"
Link

css-tricks.com: Hier finden sich öfters mal sinnvolle Tips und Tricks zu CSS – aber nicht nur ...
Link

SASS: SASS ist "CSS für Fortgeschrittene" und bietet gerade bei komplexen Seiten mit vielen Elementen aber auch viele Vorteile und erweiterte Möglichkeiten. Beispielsweise können Variablen gesetzt und vielfach verwendet werden, kleine Funktionen ("Mixins") bringen Programmlogin ins CSS. SASS-Dateien müssen kompiliert (d.h. durch ein Programm in normales CSS übersetzt) werden, damit sie für Webseiten funktionieren.
Link

Compass: ... gleichzeitig ein Compiler und eine Funktions- oder Mixin-Sammlung für SASS. Basiert auf Ruby, steuerbar via Kommandozeile.
Link

Webfonts

Eigene Schriften in der Web-Anwendung sind heutzutage kein Problem mehr

Google Fonts: Google hostet eine Vielzahl an Schriften, die man direkt in die eigene Website einbinden und so recht einfach verwenden kann.
Link

Fontsquirrel: Umfangreiche Sammlung an (brauchbaren) Free Fonts und freien Webfonts. Das Beste ist aber ein serverbasierter Konverter, in dem man selbst Schriften hochladen und in Webfont-Pakete konvertieren kann.
